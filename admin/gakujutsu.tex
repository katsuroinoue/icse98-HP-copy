\begin{center}
第２０回ソフトウェア工学国際会議の共同主催の申請について（副申Ｉ）
\end{center}

情報処理学会および日本ソフトウェア科学会の申請に係る標記国際会議につい
て、当委員会は、日本学術会議の行う国際学術交流事業の実施に関する内規
（昭和62年4月23日第102総会決定）第33条（共同主催の要件）の諸条件を満た
しており、日本学術会議が共同主催する国際会議として適切であると考えます。

\bigskip
○ 推薦理由

標記国際会議は、ＩＥＥＥ (The Institute of Electrical and Electronics
Engineers, Inc.)  およびＡＣＭ(The Association for Computing
Machinery)が母体となって毎年一度開催する、コンピュータのソフトウェアの
開発に関する諸問題について、幅広く研究発表および討議を行なうものです。
標記国際会議を日本で開催することは、ＩＥＥＥ、ＡＣＭの委員、および過去
の大会委員長やプログラム委員長から構成されるソフトウェア工学国際会議運
営委員会によって、平成７年１０月１０日正式に決定されました。その準備は、
情報処理学会および日本ソフトウェア科学会と情報学研究連絡委員会とが密接
に協力しながら以前から進められてきました。日本では1982年以来行なわれて
いません。

ソフトウェア工学の多くの研究は、過去、米国や欧州そして日本が中心になっ
て行なわれてきました。しかし、近年、アジア太平洋諸国の経済的な発展は目覚し
く、それにつれて、それらの多くの国ではソフトウェア産業の重要性を認識し、
その発展育成に努めています。日本で本会議を開くことによりアジア太平洋諸国
と欧米とを結ぶ場とすることができ、日本および世界のソフトウェア工学の研
究やソフトウェア産業へ多大な貢献を行なうものと期待されます。

以上に述べた事柄から、本会議は日本学術会議が共同主催する国際会議として
誠に適切であると判断し、是非とも共同で主催して頂くよう、お願いいたしま
す。

\begin{enumerate}

\item{会議の名称}

和文名： 第２０回ソフトウェア工学国際会議

英文名： The 20th International Conference on Software Engineering


\item {主催}

  日本学術会議（希望）、（社）情報処理学会、日本ソフトウェア科学会




\item{後援（予定）}

文部省、通産省、郵政省、科学技術庁

\item {協賛（予定）}

（社） 電子情報通信学会、人工知能学会、
  ソフトウェア技術者協会、（財）京都高度技術研究所


\item{開催時期}

平成１０年（１９９８年）４月１９日（日）−２５日（土）（７日間）

\item{開催場所}

国立京都国際会館（京都府京都市左京区宝ヶ池）

\item{会議の性格と目的}
  \begin{enumerate}
  \item{会議の種類}
   
    電気電子学会 (IEEE: The Institute of Electrical and Electronics
    Engineers, Inc.)および計算機協会(ACM: The Association for
    Computing Machinery)が共同の母体機関となって運営する国際会議。ソフ
    トウェア工学の分野では最も歴史が長く権威あるもの。

  \item {開催の周期} 

    毎年一度

  \item{開催の目的}

    \begin{enumerate}
    \item {ソフトウェア工学の内容と位置付け}
      
      近年、コンピュータは急速に普及し、その効率的な利用が現代の社会に
      不可欠なものとなっている。このような需要が高まるにつれ、これらの
      コンピュータを動かすためのソフトウェアの欠陥や開発の遅れが社会に
      与える損失も大きくなってきている。ソフトウェアを間違いのないよう
      に、かつ効率的に作る技術がますます重要になってきている。

      ソフトウェア工学とは、1968年にNATOの技術委員会主催の会議で初めて
      用いられた用語で、これらソフトウェアの開発に関する諸問題を解決す
      るためいろいろな研究分野から構成されている。 例えば、効率的なソ
      フトウェア開発を行なうための方法論、作成されたソフトウェアの信頼
      性を調べるためのテスト手法、さらに開発の支援を行なうシステム作成
      などは代表的な研究対象である。

      ソフトウェア工学は、他のコンピュータ関連の分野と密接な関わりがあ
      る。例えば、最新のコンピュータハードウェアやネットワークは、ソフ
      トウェア開発の形態を変えつつあり、それに応じた研究成果も発表され
      てきている。また、コンパイラ技術のように、ソフトウェア工学の一分
      野から分かれ 独立した研究分野に発展する場合もある。
      
      今後、コンピュータは一般家庭にも広く普及し、ますますソフトウェア
      の重要性が増していくものと考えられる。ソフトウェア産業は、各国に
      とって非常に重要なものとなってきており、ソフトウェア工学は、それ
      らの発展に必須な学問として大きな役割を果たしていくものと期待され
      ている。

    \item {ソフトウェア工学の現状}

      今日、ソフトウェア工学は、下記のような研究分野を対象にし、基礎的
      な理論研究から現実のソフトウェア開発に適用する応用研究まで幅広い
      活動を行なっている。

     \begin{itemize}
     \item 代数的な記述など、ソフトウェアの形式的仕様記述法の提案とそ
       れを用いた信頼性向上技術

     \item クライアント／サーバシステムを始めとしたソフトウェアの構造
       の整理方法と新しい構造の提案

     \item ソフトウェアの大きさやそこに含まれる欠陥の数など、ソフトウェ
       アの量的な評価方法とその適用

     \item ソフトウェア開発の作業プロセスの記述とプロセスに基づいた開
       発支援

     \item 効率的なソフトウェア開発作業を行なうためのプログラミング環
       境

     \item ネットワークやマルチメディアツールを利用した開発支援環境の
       構築、評価

     \item 種々のソフトウェアのテストの手法とその評価方法

    \end{itemize}

  \item {会議開催の目的}

    ソフトウェア開発に関する種々の課題について、世界各国のソフトウェア
    工学の研究者からソフトウェア開発の実務者まで一堂に会して、最新の研
    究成果の発表や討議を行なう。これによって、ソフトウェア工学の研究の
    発展のみならず、現実のソフトウェア開発の効率向上やソフトウェア製品
    の欠陥減少を目指し、社会におけるコンピュータ利用のより健全な発展を
    目指す。

  \item {会議による期待される効果}

      \begin{itemize}
      \item 最新のソフトウェア工学に関するのいろいろな研究成果が発表さ
        れるので、新たな研究の方向づけやソフトウェア産業の戦略決定を通
        じて、今後のソフトウェア生産の拡大に大きな貢献をすると思われる。

      \item いろいろな国の研究者や実務者が一堂に会して意見交換や議論を
        行ない、人的な交流を深めることは、わが国のみならず各国の研究者、
        実務者の国際的視野を広げ、世界的な研究や生産活動が推進される。

      \end{itemize}
      
    \end{enumerate}


  \item {ソフトウェア工学国際会議運営委員会、情報処理学会、日本ソフト
      ウェア科学会と日本学術会議との関連および経緯}

    ソフトウェア工学国際会議の最高意思決定機関であるソフトウェア工学国
    際会議運営委員会は、IEEEおよびACMからの委員の他、過去のソフトウェ
    ア工学国際会議の委員長やプログラム委員長から構成されている。 この
    委員会には、日本から鳥居宏次（奈良先端科学技術大学院大学）、および
    片山卓也（北陸先端科学技術大学院大学）が参加しており、過去のプログ
    ラム委員長を引き受けるなど、積極的に活動している。この両名は、学術
    会議の土居範久会員と、準備委員会の発足や国内学会の主催の申請などに
    関して緊密に連絡対応を行なっている。また、共同主催する社団法人情報
    処理学会と日本ソフトウェア科学会の両学会は、第４部の情報学研究連絡
    委員会や第５部の情報工学研究連絡委員会等に関係する日本学術会議の登
    録学術団体であり、ソフトウェア工学に関する活動を総合的に行なってい
    るものである。

\pagebreak

  \item {情報処理学会、日本ソフトウェア科学会とソフトウェア工学国際会
      議運営委員会との関連}

    ソフトウェア工学国際会議運営委員会の母体となるIEEEおよびACMは、ア
    メリカを中心とし、世界的に多くの会員を組織している学術団体である。
    一方、情報処理学会、日本ソフトウェア科学会は、日本国内を中心とした
    学術団体である。これらは、それぞれが独立した組織であるが、会員や役
    員の多くが重複していたり、多くの国際会議を共催するなど、非常に緊密
    な関係にある。実際1982年に東京で開催された第６回ソフトウェア工学国
    際会議は、IEEE、ACMと情報処理学会とが共同主催した。ソフトウェア工
    学国際会議運営委員会の委員である片山卓也は日本ソフトウェア科学会の
    前理事長であり、また、鳥居宏次は、情報処理学会の前編集委員であり、
    これらの学会との意思疎通をはかってきた。情報処理学会および日本ソフ
    トウェア科会では、第２０回ソフトウェア工学国際会議を共同主催するこ
    とをそれぞれの理事会で決定している。

  \end{enumerate}


\item {日本開催の経緯と意義}

  \begin{enumerate}
  \item {日本開催の要望及び決定の状況}

    この国際会議は、1982年9月、東京で第６回大会を開催して以来、日本で
    は行なわれてこなかった。近年のアジア太平洋諸国の経済的な発展に伴っ
    て、日本での開催を期待されてきていたが、同じ場所で２度開催しないと
    いう方針があり、東京以外の場所では不便であるため日本での開催が見送
    られていた。 昨年、関西新空港が開港し、海外から訪問しやすくなった
    ため、関西での開催を引き受けることにし、1995年10月10日のソフトウェ
    ア工学国際会議運営委員会において京都での開催が決定した。


  \item{過去の開催状況}

    過去における開催状況を下表で示す。第９回大会までは、１年半毎、それ
    以降は毎年１回開催されている。最近の開催地は、米国西部地域、米国以

\item {過去の日本での開催}

  1982年9月、東京で第６回大会を開催した。それ以外はない。

\item {日本での開催の意義}

  ソフトウェア工学の多くの研究は、米国や欧州あるいは日本で行なわれてき
  ていた。しかし、近年、アジア太平洋諸国の経済的発展は目覚しい。それに
  つれ、それらの多くの国で、ソフトウェア産業の重要さに留意し、その育成
  発展に努めている。 韓国、シンガポール、台湾、オーストラリアなどの国
  では、すでに多くのソフトウェア工学関連の研究が遂行されており、その研
  究成果が徐々に発表されつつある。この国際会議を日本で開催することによ
  り、多数のアジア太平洋地域の研究者の参加が見込まれ、今まで欧米中心で
  あったソフトウェア工学研究の国際交流をより世界的なものにすることがで
  きる。

  また、日本のソフトウェア工学の研究者やソフトウェア開発の実務者にとっ
  ては、最先端のソフトウェア工学研究の情報が得られるとともに、他国の参
  加者との交流を通じて視野を広げることができ、今後の日本の研究や産業の
  発展に大いに役立つと各界から期待されている。

\item {今回の開催の意義}

  ソフトウェア工学をとりまく環境は大きく変わりつつある。今までのような
  大型コンピュータを主体とした文字や数字の計算システムに代わって、小型
  でネットワークで継れた小型分散コンピュータを主体とした音声や画像を扱
  うことができるいわゆるマルチメディアシステムが、日常的に使われるよう
  になってきた。それにつれてソフトウェアの開発形態も多様化、複雑化して
  おり、それらに起因する問題を有効に解決するソフトウェア工学の技術が緊
  急に求められている。本会議ではそれらのテーマを重点的な議題として議論
  を予定しており、多くの成果が期待される。

  \end{enumerate}
